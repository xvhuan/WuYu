# 吾语 · WuYU

轻量的语录收集站，支持截图上传、时间轴浏览以及后台编辑，适合个人或小团队快速搭建私有语录墙。

![img](https://img.qaq.al/i/2025/11/16/acfacb6706ed2bbafe526661d538ca92.)

## 功能亮点

- **流畅前端体验**：无限滚动时间轴、截图缩略展示 + 查看原图，支持粘贴/拖拽上传。
- **安全机制**：上传密码、首页访问锁、后台账号可在线配置，支持 IP 封禁与自定义后台路径。
- **后台管理**：搜索/编辑/删除语录，替换或清除截图，实时调整网站名称、字体大小等样式。
- **可配置项**：所有个性化设置持久化于 `data/settings.json`，无需改代码即可生效。

## 快速开始

```bash
npm install
npm start
# 默认监听 34212 端口
```

首访首页可在后台设置中开启/关闭访问密码，默认密码`wuyu123`；后台路径默认 `/admin`，也可随时修改（记得同步更新访问入口）。

## 目录结构

```
public/        # 前端页面与样式
server.js      # Express 服务，含 API 与设置逻辑
uploads/       # 截图文件（自动创建）
data/          # quotes.json / settings.json
```
